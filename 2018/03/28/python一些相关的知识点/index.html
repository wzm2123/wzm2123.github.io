装饰器

定义：本质是函数，（装饰其他函数）就是为其他函数添加附加功能

原则：

  1. 不能修改被装饰的函数额源代码
  2. 不能修改被装饰的函数的调研方式

实现装饰器知识储备：

  1. 函数即“变量”
  2. 高阶函数
    a: 把一个函数名当做实参传给另一个函数（在不修改被装饰函数源代码的情况下为其添加功能）<br>
b: 返回值中包含函数名（不修改函数的调用方式）

标准库

time与datetime

  1. time.time() 获取当前时间戳
  2. time.sleep() 程序睡几秒
  3. time.gmtime(时间戳) 把秒转换成UTC当前时间(元组形式)
  4. time.localtime(时间戳) 把秒转换成本地当前时间(元组形式)
  5. time.mktime() 当前时间转换成时间戳
  6. time.strftime(“%Y-%m-%D %H:%M:%S”,时间元组格式变量)
    把元组转换格式化字符串输出时间

  7. time.strptime(时间元组格式变量,%Y-%m-%D %H:%M:%S)
    把字符串转换元组输出

  8. datetime.datetime.now() 获取当前时间

random模块

  • random.random() 随机浮点数（0到1之间）
  • random.randint(1,3) 随机整数（1到3之间 包括1和3）
  • random.randrange(1,3) 随机整数（1到3之间 不包括3）
  • random.chioce(“hello”) 随机取值
  • random.sample(“hello”,2) 随机2位取值
  • random.uniform(1,3) 随机浮点数（1到3之间）
  • 洗牌功能
    eg:
    l=[1,2,3,4]

    random.shuffle(l)

    =》l=[2,3,4,1]

os模块

  • os.getcwd() : 得到当前工作目录，即当前python脚本工作的目录路径。
  • os.chdir(r”dirname”) : 改变工作目录到dirname
  • os.makedirs(‘dirname1/dirname2’) 可生成多层递归目录
  • os.removedirs(‘dirname1’) 若目录为空，则删除，并递归到上一级目录，如若也为空，则删除，依此类推
  • os.mkdir(‘dirname’) 生成单级目录
  • os.rmdir(‘dirname’) 删除单级空目录，若目录不为空则无法删除，报错
  • os.listdir(‘dirname’) 列出指定目录下的所有文件和子目录，包括隐藏文件，并以列表方式打印
  • os.remove() 删除一个文件
  • os.rename(“oldname”,”newname”) 重命名文件/目录
  • os.stat(‘path/filename’) 获取文件/目录信息
  • os.sep 输出操作系统特定的路径分隔符，win下为”\“,Linux下为”/“
  • os.linesep 输出当前平台使用的行终止符，win下为”\r\n”,Linux下为”\n”
  • os. name 输出字符串指示当前使用平台。win->’nt’; Linux->’posix’
  • os.path.abspath(path) 返回path规范化的绝对路径
  • os.path.split(path) 将path分割成目录和文件名二元组返回
  • os.path.dirname(path) 返回path的目录。其实就是os.path.split(path)的第一个元素
  • os.path.basename(path) 返回path最后的文件名。
  • os.path.exists(path) 如果path存在，返回True；如果path不存在，返回False

re模块

常用正则表达式符号

  • ‘.’ 默认匹配除\n之外的任意一个字符
  • ‘^’ 匹配字符开头
  • ‘$’ 匹配字符结尾
  • ‘+’ 匹配前一个字符1次或多次，re.findall(“ab+”,”ab+cd+abb+bba”) 结果[‘ab’, ‘abb’]
  • ‘|’ 匹配|左或|右的字符，re.search(“abc|ABC”,”ABCBabcCD”).group() 结果’ABC’
  • ‘\d’ 匹配数字0-9

最常用的匹配语法

  • re.match() 从头开始匹配
  • re.search() 匹配包含
  • re.findall() 把所有匹配到的字符放到以列表中的元素返回
    (没有group()方法)
