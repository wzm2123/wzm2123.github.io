css的使用

  • rgb颜色对照表

  • 行内标签无法设置高度，宽度，边距

    css选择器

    优先级：标签上的style优先，编写顺序，越往下越优先

多个文件同时引用某种属性，先新建css文件

边框

宽度 样式 颜色

  • height
    width(可以百分号表示)
    line-height水平居中（但数值必须等于height）
    color（内容颜色）
    font-size（字体大小）
    font-weight(字体粗细)
    text-aligan:center(内容左右居中)

float：让标签浪起来，块级标签也可以堆叠

父类强制把浮起来的子类又压下来—-》用来显示父类边框

display

  • display: inline;————》把块级标签变为行内标签
  • display: block;—–》把行内标签变为块级标签
  • display: inline-block;—–>具有行内标签的属性，默认自己有多少占多少。具有块级属性，可以设置高度，宽度，边距。
  • display: none;—->让标签消失

padding margin：0 auto；–边距，内边距，外边距

  • margin-top： 子类本身属性不变，上部距离父类变化
  • padding-top： 子类高度发送变化，从上增加子类额外高度

position : fixed; —————》固定在页面的某个位置

  • top:0;
  • right:0;
  • bottom:0;

    position : relative；（父类）+

    position : absolute；（子类） —> 固定相对父类的位置

  • opacity:0.5; 透明度（0-1）
  • z-index:10; 谁大谁在上面
  • overflow:auto;图片过大时出现滚动条
  • overflow:hidden;图片过大时隐藏

hover

  • . abc : hover{
    属性内容<br>
    } 只有当鼠标放上该标签时才会启动该属性


背景图片

  • background-image:url(‘图片地址’); # 默认，div大，图片重复放
  • background-repeat:no-repeat/repeat-x/repeat-y; # 不重复放图片，只水平加，只竖直加

  • background-position-x:10px; #只移动背景图片

    background-position-y:10px;
