linux操作方法

  • ls —显示当前路径下的文件以及文件夹

  • pwd —-显示当前路径

  • cd

  • touch 文件名 ——– 创建文件
  • touch .文件名 ——–创建隐藏文件
  • mkdir 文件夹 ——–创建文件夹
  • clear —————–清屏
  • ls / ——————-显示根目录
  • ls /bin —————显示bin目录
  • ls -选项（a，l，h） 显示全部文件/以列表显示/(l,h)更科学显示内容大小(可连写)
  • ls 2*—————-通配符（2?）
  • ls 1[12345]3.txt———选择取
  • ls > xxx.txt————–重定向
  • ls >> xxx.txt———–追加
  • 命令 –help ===== man 命令 (按Q退出，按F翻屛，按B回翻屏)
  • gedit 部分文件名+“tab”——–自动补全（可以编辑）
  • cat 部分文件名+“tab”———–自动补全，只查看内容（）
  • history————————–历史命令
  • ！200——-执行200对应的命令
  • rm 文件或文件夹名———-删除
